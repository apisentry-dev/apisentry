## Install

<CodeGroup>

```bash macOS / Linux
curl -L https://github.com/apisentry-dev/apisentry/releases/latest/download/apisentry-linux-amd64 \
  -o apisentry
chmod +x apisentry && sudo mv apisentry /usr/local/bin/
```

```bash Windows
# Download apisentry-windows-amd64.exe from the latest release:
# https://github.com/apisentry-dev/apisentry/releases/latest
```

```bash Build from source
git clone https://github.com/apisentry-dev/apisentry
cd apisentry
go build -o apisentry .
sudo mv apisentry /usr/local/bin/
```

</CodeGroup>

## Run your first scan

You need two things: an OpenAPI spec and a running target API.

```bash
apisentry scan --spec openapi.yaml --target https://api.yourapp.com
```

## Authenticate your scan

If your API requires a Bearer token:

```bash
apisentry scan \
  --spec openapi.yaml \
  --target https://api.yourapp.com \
  --token eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

APISentry uses this token for authenticated requests. It also **deliberately tests unauthenticated access** to detect broken auth vulnerabilities.

## Save a report

<CodeGroup>

```bash HTML report
apisentry scan \
  --spec openapi.yaml \
  --target https://api.yourapp.com \
  --format html \
  --output report.html
```

```bash JSON
apisentry scan \
  --spec openapi.yaml \
  --target https://api.yourapp.com \
  --format json \
  --output results.json
```

```bash SARIF (GitHub Code Scanning)
apisentry scan \
  --spec openapi.yaml \
  --target https://api.yourapp.com \
  --format sarif \
  --output results.sarif
```

</CodeGroup>

## Enable AI analysis

AI analysis uses Claude Haiku to review each finding, remove false positives, and add remediation advice:

```bash
ANTHROPIC_API_KEY=sk-ant-... apisentry scan \
  --spec openapi.yaml \
  --target https://api.yourapp.com \
  --ai
```

<Info>
AI analysis costs ~$0.002 per finding. A typical scan with 20 findings costs under $0.05.
</Info>
