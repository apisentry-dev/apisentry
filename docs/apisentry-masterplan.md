## Часть 1: Обзор проекта

### 1.1 Суть

**APISentry AI** — developer-first платформа для автоматического тестирования безопасности API с помощью AI. Загружаешь OpenAPI spec (или даёшь URL), и AI генерирует сотни контекстных тест-кейсов: BOLA, broken auth, injection, mass assignment, rate limiting, SSRF и другие OWASP API Top 10 уязвимости. Встраивается в CI/CD (GitHub Actions, GitLab CI, Jenkins). Стоит $49/мес вместо $50K+/год у enterprise-решений.
